# DealOs v2.0

DealOs v2.0 is a React + Vite application for managing and analyzing real estate deal listings.

## Tech Stack

- React 19
- TypeScript
- Vite
- Gemini API (`@google/genai`)

## Local Development

Prerequisites:

- Node.js 18+

1. Install dependencies:
   `npm install`
2. Configure environment variables in `.env.local`:
   `GEMINI_API_KEY=your_api_key_here`
3. Start the dev server:
   `npm run dev`
4. Open:
   `http://localhost:5173`

## Build

- Production build:
  `npm run build`
- Preview build:
  `npm run preview`
